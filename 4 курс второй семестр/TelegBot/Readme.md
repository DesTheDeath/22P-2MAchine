## Используемое API

Бот использует встроенное API **KoboldCPP**.  KoboldCPP предоставляет удобный интерфейс для взаимодействия с LLM моделями. Подробнее можно узнать здесь: [https://github.com/LostRuins/koboldcpp](https://github.com/LostRuins/koboldcpp)

## Модель ИИ

В качестве модели искусственного интеллекта используется **Gemma-3-27B-Abliterated**. Это мощная языковая модель, разработанная Google. 

## Модель Трансформера и Решаемая Задача

Модель трансформера – **mmproj**. Эта модель хорошо подходит для решения задач генерации текста и понимания контекста. Она способна генерировать связные и осмысленные ответы на различные вопросы и запросы.
